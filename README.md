# PERFY — Engineering High-Performance Businesses

PERFY is a performance-empowerment ecosystem built to transform businesses into structured, scalable, and high-performing organisations. This portal serves as the digital gateway to PERFY's suite of intelligence assessments and operational management modules.

## 🚀 Key Features

- **Individual Diagnostic Engines**: Structured personality & intelligence assessment portal mapping potential to impact.
- **Enterprise Modules Catalog**: Modular operational suite containing workflow builders, role hierarchy management, strategy metrics (OKRs), and inventory/CRM panels.
- **Operator Admin Console**: Dynamic control deck to adjust pricing structures, toggle modules, upload sample reports, and manage contact requests.
- **Premium Cinematic UI**: Warm-cream and deep-maroon themed glassmorphic design system utilizing Framer Motion, dynamic particle fields, and tailored typography.

## 🛠️ Tech Stack

- **Frontend Core**: React 18 (TypeScript), Vite
- **Styling**: Tailwind CSS
- **Transitions**: Framer Motion
- **Services**: Tanstack React Query, Supabase Client
- **Document Previews**: React-PDF / PDFjs

## 📦 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed.

### Installation

```bash
# Install dependencies
npm install

# Run the local development server
npm run dev

# Create a production build
npm run build
```
